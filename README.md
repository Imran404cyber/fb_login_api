# 📖 About

This repository is a research and educational project focused on understanding how modern web authentication works.

It explores browser requests, HTTP headers, cookies, CSRF protection, session handling, and authentication flow from a learning perspective.

---

# ✨ Features

- ✅ HTTP Request Analysis
- ✅ Login Flow Research
- ✅ Cookie Inspection
- ✅ Header Analysis
- ✅ CSRF Token Study
- ✅ Session Management
- ✅ Browser Request Comparison
- ✅ Authentication Flow Learning
- ✅ Request Sequence Analysis
- ✅ Network Debugging
